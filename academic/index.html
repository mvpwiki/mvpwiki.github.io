Academic

신입생 및 재학생이 학사에 관련된 것을 귀찮게 찾아보지 않도록 메뉴얼화하여 정리한 페이지입니다.

졸업 요건

대부분 졸업에 관한 필요사항은 오리엔테이션 때 알려주거나 과 홈페이지 링크를 참고하면 되지만, 더 이해하기 쉽게 핵심을 정리해 두도록 하겠습니다.

우선 졸업에 필요한 과정을 순차적으로 표현하면 다음과 같습니다.

  1. Course Work를 수료하였는가
  2. 종합시험 and 영어시험을 통과하였는가
  3. 연구지도를 수강하고 연구계획서를 제출하였는가
  4. 졸업에 필요한 논문 요건 을 충족하였는가
  5. 학위논문을 작성하고 예비심사, 본심사를 통과하였는가

하나씩 살펴보도록 하겠습니다.

개별지도연구

여기서 개별지도연구는 실제 수업을 진행하는 과목은 아니고 지도교수가 학점을 평가하게 됩니다.

석사는 개별지도연구 1, 2, 3을, 박사는 4, 5, 6을, 통합과정은 1, 2, 3, 4를 학기별로 순차적으로 수강신청하면 됩니다.


공학윤리와 연구방법론

공학윤리수업은 청강과목으로 일반대학원의 공과대학공통 항목에서 0학점으로 수강할 수 있습니다.


비전공과목과 전공인정

비전공과목은 전기전자공학 소속이 아닌 과목을 의미하며, 학기가 마치고 공지가 있을 경우 전공과목 인정 신청을 통해서 전공과목으로 인정받을 수 있습니다. (최대 6학점)

정규학기란 석사 1~4학기 / 박사 1~4학기 / 통합 1~6학기를 의미하며, 석사와 박사는 3학기 내에 수업을 다 들어야 합니다

2. 종합시험 및 영어시험

입학원서를 제출할 경우 또는 매 방학마다 종합시험 및 영어시험 응시가 가능합니다.

수업을 다 수강하고, 종합시험과 영어시험을 모두 통과해야 “수료” 상태가 됩니다

단, 전문연구요원을 지원하려는 분께서는 수료를 해서는 안 됩니다.


영어시험

실제 영어시험은 아니며 TOEIC, TEPS, TOEFL의 유효기간이 남아 있는 성적표를 제출하면 됩니다.

1학기부터 응시가 가능하므로 최대한 빨리 완료하는 것을 권장합니다.

해당 점수를 만족하면 통과합니다.

시험 점수
TOEIC 730
TEPS 600
TOEFL 207 (CBT)


종합시험

종합시험도 실제 시험은 아니며, 본인이 수강한 과목 중 일부를 선별하여 평균학점 3.65을 넘으면 됩니다.

석사 3학기, 박사 3학기, 통합 5학기 이상부터 응시 가능합니다.

과정 과목수 비고
석사과정 3과목 전공, 지도교수 2과목 가능
박사과정 3과목 전공, 지도교수 2과목 가능
통합과정 6과목 전공, 지도교수 2과목 가능

단, C+ 이하 과목은 선택할 수 없습니다.

박사과정의 경우 동 대학원 석사과정 때 응시한 과목을 선택할 수 없습니다.
