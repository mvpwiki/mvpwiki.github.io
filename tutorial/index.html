Tutorial


연세대학교 영상인식연구실 Tutorial (최종 업데이트: 2019-1학기 기준, 관리자 김태오)

본 과정은 MVPLAB에 입학하는 모든 신입생이 입학 전 필수로 이수하여야 하는 과정입니다.



Prerequisites


MATLAB

MATRIX 연산 기반 수치 계산 Tool. Image Processing에 적합. 연세대학교에서는 MATLAB 정품을 무료로 제공. 설치는 간단 설치 가이드를 참고 , 문제 발생시에는 정규 설치 가이드를 참고.


Anaconda

Python Library Package로써, Numpy, Pylab, Pandas, Matplotlib 등 대부분의 핵심 Library를 지원. 다운로드는 Anaconda 홈페이지에서 Python 3.6 기준 Windows 64비트 기준으로 받으면 된다. 설치 시에는 Default로 설정. Anaconda는 Visual Studio같은 개발 환경이 아니라 단순히 Source Code만 제공. 콘솔에서는 실행 가능.


Jupyter Notebook (IPython)

Python 개발 환경 툴(IDE)로써, Anaconda를 설치하면 자동으로 포함되어 있음. Jupyter Notebook은 Python을 MATLAB처럼 Interactive하게 실행시켜 주는 개발 환경(IDE)라고 생각하면 된다. 간단 사용법을 참고.


Pycharm

Build와 Debugging을 통해서 Project 형식으로 관리할 수 있는 Python 개발 환경. 설치법은 홈페이지에 들어간 뒤, 설치 및 설정 방법를 참고.







Tutorial Tracks





Track 1: Basic Matrix and Image Processing

  • MATLAB 설치 필요 (교내 무료)
  • 교재 Hard Copy또는 pdf 필요

MATLAB의 기본적인 사용법과 기본적인 Image Processing 알고리즘 및 코딩에 대해 학습.

배우는 내용: Images, Point Processing, Filtering, Noise, Edges, Frequency Processing, Hough and Distance Transform, Morphology


Track 2: Basic Python

  • Anaconda 설치 필요
  • Jupyter Notebook 사용 권장 (Pycharm 가능)
  • 소스 코드
  • 더 기초가 필요할 경우

Python 기초 및 기본 Library 학습

배우는 내용: Numpy, Matplotlib만 학습


Track 3: Deep Learning

  • 메인 Website
  • 가입 및 무료 수강 방법

영어 강의, 수강하면서 한글로 요약 PPT 제작하여 발표, Youtube Version은 한글 자막 제공

배우는 내용 1: Logistic Regression, Gradient Descent, Neural Networks, Backpropagation

배우는 내용 2: Regularization, Initialization, Optimization, Hyperparameter Tuning and Normalization

배우는 내용 3: Metric, Evaluation, Bias and Variance, Transfer and Multi-task Learning, E2E Learning

배우는 내용 4: CNNs, ResNets, Inception Networks, Object Detection, Face Recognition, Style Transfer

배우는 내용 5: RNNs, Word Embeddings, Attention Models


Track 4: Deep Learning with Tensorflow and PyTorch

  • Anaconda, Pycharm, Tensorflow, PyTorch 설치 필요
  • Pycharm 사용
  • 반드시 Linux 원격 서버 SSH 기반 가상환경으로 코딩할 것. 메뉴얼 링크
  • Tensorflow 강의 및 강의자료 Website
  • Tensorflow 소스 코드
  • PyTorch 강의 및 강의자료 Website
  • PyTorch 소스 코드

Python Deep Learning Library인 Tensorflow와 PyTorch를 Machine Learning 예제를 통해 학습


Track 5: Deep Research

  • 세미나 형식
  • 중간 과제: 머신러닝 알고리즘 구현 1개, 희망 분야에서 기초 알고리즘 구현 1개
  • 최종 과제: 신입생 세미나, 최신 논문 1편을 선정하여 읽고 정리하여 발표

저널/학회 소개 및 논문검색 및 논문읽기 메뉴얼 정독
