Prerequisites


MATLAB

MATRIX 연산 기반 수치 계산 Tool. Image Processing에 적합. 연세대학교에서는 MATLAB 정품을 무료로 제공. 설치는 간단 설치 가이드를 참고 , 문제 발생시에는 정규 설치 가이드를 참고.


Anaconda

Python Library Package로써, Numpy, Pylab, Pandas, Matplotlib 등 대부분의 핵심 Library를 지원. 다운로드는 Anaconda 홈페이지에서 Python 3.6 기준 Windows 64비트 기준으로 받으면 된다. 설치 시에는 Default로 설정. Anaconda는 Visual Studio같은 개발 환경이 아니라 단순히 Source Code만 제공. 콘솔에서는 실행 가능.


Jupyter Notebook (IPython)

Python 개발 환경 툴(IDE)로써, Anaconda를 설치하면 자동으로 포함되어 있음. Jupyter Notebook은 Python을 MATLAB처럼 Interactive하게 실행시켜 주는 개발 환경(IDE)라고 생각하면 된다. 간단 사용법을 참고.


Pycharm

Build와 Debugging을 통해서 Project 형식으로 관리할 수 있는 Python 개발 환경. 설치법은 홈페이지에 들어간 뒤, 설치 및 설정 방법를 참고.


Visual Studio

C, C++을 Main으로 사용하게 될 개발 환경. 커뮤니티 버전 다운로드는 무료.


OpenCV

Open Source Computer Vision Library. C++기반, 설치는 교재 참고


Tensorflow

Deep Learning Library, 필요시 설치


Pycharm

Deep Learning Library, 필요시 설치
