Visual Tracking

발표자: 이현성

발표 일자: 2018년 5월 4일



Paper 1: ECO: Efficient Convolution Operators for Tracking

Paper 2: Learning Multi-domain Convolutional Neural Networks for Visual Tracking



Object tracking은 영상 내에 물체의 초기 위치만 주어진 상태에서 대상의 궤적을 추적하는 computer vision의 고전적인 problem입니다. 그중 visual object tracking(VOT)은 image sequence 내의 한가지 물체를 추적하는 문제로, 매우 제한된 훈련 샘플 집합으로부터 occlusion이나 deformation 등의 문제 대처가 가능한 일반화(generalize)된 대상의 외형 모양을 찾을 수 있어야 하는 어려움이 있습니다. VOT가 고전적인 problem인 만큼, 이번 세미나에서는 VOT의 문제를 과거부터 현재까지 어떠한 방식으로 풀어 왔는지에 크게 2가지 방법(Correlation Filter, Deep Learning)에 대해 survey 형식으로 진행하려 합니다.


1. Correlation Filter 기반 알고리즘 소개

  • Visual object tracking using adaptive correlation filters, CVPR 2010
  • A scale adaptive kernel correlation filter tracker with feature integration, ECCV 2014
  • Learning spatially regularized correlation filters for visual tracking, CVPR 2015
  • Beyond correlation filters: Learning continuous convolution operators for visual tracking, ECCV 2016
  • Eco: Efficient convolution operators for tracking, CVPR 2017

2. Correlation Filter 사용하지 않은 Deep Learning 기반 알고리즘 소개

  • Learning multi-domain convolutional neural networks for visual tracking, CVPR 2016
  • Fully-convolutional siamese networks for object tracking, ECCV 2016
  • End-to-end representation learning for correlation filter based tracking, CVPR 2017
